# Representative images of commercial powder-dispensing platforms

One representative photograph per platform named in
[`../commercial-powder-dispensing-landscape.md`](../commercial-powder-dispensing-landscape.md)
and the second-batch Edison queries in
[`../edison_queries.json`](../edison_queries.json).

Each image is a representative product photograph fetched from the
vendor's own website where possible, and otherwise from a reseller,
trade-press, or industry directory page that hosts the vendor's marketing
photo. Images are stored here for **non-commercial research and editorial
reference only** to support the comparative analysis in this repository;
all rights remain with the respective vendors / photographers.
Source URL and originating page for each file are listed below and in
[`_manifest.json`](./_manifest.json).

If you are the rights-holder for any image and would like it removed or
replaced, please open an issue.

Total images: **40**.

A composite figure of all 40 panels (with (a)–(an) labels and a caption
keyed to the table below) is available at
[`mosaic.png`](./mosaic.png) and [`mosaic_caption.md`](./mosaic_caption.md).

## Reproducing the fetch

The images were collected with a small `ddgs` (DuckDuckGo)
image-search script that constrained results to vendor / reseller
domains and downloaded only `image/*` responses larger than ~12 KB.
Suspicious hits (careers banners, generic stock photos) were re-fetched
with stricter domain allow-lists; the MTI Mixer image was pulled
directly from `mti-mixer.de` after the auto-detected `mtimixer.com`
turned out to be a dead host.
